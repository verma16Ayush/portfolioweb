MusicSense0.5
  • Python

    • Musicsense 0.5 is an audio fingerprinting Console Application based on Python

    The Project was originally made in a team of four at NITH's intra college hackathon Hack4.0 under mentorship of a final year senior.

    This is my own implementation of the entire project. MusicSense0.5 aims at predicting a song's name by listening to a short clip of it. The project works by creating its own database of peaks in the song relative to other peaks using hashing algorithms and then checking for hash collisions between the listened clip and the database

    Tools and Tech used - Python, image processing functionalities of the SciPy library, SQLite3, PyAudio etc.
